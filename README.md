# Login Page (HTML + CSS)

Minimal static login page built with `index.html` and `style.css`.

## Files

- `index.html` — markup for the login page. Contains two boxes (`#box1` and `#box2`); `#box2` holds the login form.
- `style.css` — styles for layout, card (`#box2`) and form controls.

## Prerequisites

- A modern browser (Chrome, Edge, Firefox).
- (Optional) A simple static server for testing local requests.

## How to run

1. Open the page directly in your browser:

	- Double-click `index.html` or right-click → Open With → your browser.


	- Using VS Code: install the Live Server extension and click "Go Live".

## Usage notes

- Placeholders are styled using `::placeholder` in `style.css`.
- To make images fill their parent, use `width: 100%`, `height: 100%`
